## 🚀 树莓派部署步骤（3步）

### 步骤1: 安装 mpg123（树莓派老系统）
```bash
sudo apt update
sudo apt install mpg123 -y
```

### 步骤2: 安装 Python 依赖
```bash
cd /home/xiaohehe/A_theme_one/ui/backend
pip3 install -r requirements.txt --user
# 或使用虚拟环境（推荐）
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 步骤3: 启动服务
```bash
cd /home/xiaohehe/A_theme_one/ui/backend
uvicorn server:app --host 0.0.0.0 --port 8000
```

**或者使用快速部署脚本：**
```bash
cd /home/xiaohehe/A_theme_one/ui/backend
chmod +x deploy_raspberry.sh
./deploy_raspberry.sh
```

## 📦 需要部署哪些文件？

### ✅ 需要复制到树莓派：

**后端（必须）：**
- `backend/server.py`
- `backend/music/` 文件夹（包含所有 .mp3 文件）
- `backend/requirements.txt`（可选，方便安装依赖）

**前端（如果需要在树莓派上运行前端）：**
- 整个 `ui/` 文件夹（或只复制前端部分）

### ❌ 不需要部署：
- `node_modules/`（前端依赖，可在树莓派上重新安装）
- `__pycache__/`（Python 缓存）
- `.git/`（Git 仓库）

## 🔄 部署方式

### 方式一：直接复制整个文件夹（开发阶段推荐）

使用 `scp` 或 SFTP 工具（如 WinSCP、FileZilla）：
```bash
# 从本地复制到树莓派
scp -r ui/ xiaohehe@192.168.31.159:/home/xiaohehe/A_theme_one/
```

### 方式二：只复制后端文件（生产环境推荐）

```bash
# 只复制后端
scp -r ui/backend xiaohehe@192.168.31.159:/home/xiaohehe/A_theme_one/ui/
```

## 🌐 前端配置

如果前端也在树莓派上运行，需要修改 API 地址：

在 `ui/api.ts` 中，默认会使用环境变量，或者直接修改：
```typescript
const BASE_URL = import.meta.env.VITE_ORANGE_PI_API || "http://192.168.31.159:8000";
```

创建 `.env.production` 文件：
```bash
VITE_ORANGE_PI_API=http://192.168.31.159:8000
```

## 🎯 播放器优先级

代码会自动按以下优先级选择播放器：

1. **Windows 开发环境**：
   - mpv.exe (`D:\software\mpv\mpv.exe`)
   - 系统默认播放器（如果没有 mpv）

2. **Linux 环境（树莓派/香橙派）**：
   - mpv（如果安装，香橙派推荐）
   - mpg123（树莓派老系统推荐，**当前使用**）
   - ffmpeg（如果安装了）

## 🔮 未来迁移到香橙派

**无需修改代码！** 只需：
1. 在香橙派上安装 mpv：`sudo apt install mpv -y`
2. 代码会自动检测并使用 mpv（优先级高于 mpg123）
3. 直接运行即可
