# 前端部署方案

## 当前情况
- ✅ 后端已正常运行在香橙派：`http://192.168.31.159:8000`
- ❌ 前端显示不正常（只有黑色和字母）

## 方案一：本地开发前端 + 远程后端（推荐开发阶段）

### 步骤1: 在你的电脑上运行前端

```bash
# 在你的电脑上，进入 ui 目录
cd D:\A_theme_one\ui

# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run dev
```

### 步骤2: 配置 API 地址

创建 `.env.local` 文件（在 `ui/` 目录下）：

```bash
VITE_ORANGE_PI_API=http://192.168.31.159:8000
```

或者直接修改 `api.ts`（已经自动适配，但可以手动指定）

### 步骤3: 访问前端

在你的电脑浏览器中访问：
```
http://localhost:3000
```

前端会自动连接到香橙派的后端 API。

## 方案二：在香橙派上部署前端（生产环境）

### 步骤1: 在香橙派上安装 Node.js 和 npm

```bash
# 检查是否已安装
node --version
npm --version

# 如果没安装，安装 Node.js（推荐使用 nvm 或直接安装）
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### 步骤2: 在香橙派上构建前端

```bash

cd /home/orangepi/Desktop/theme1/ui

# 安装依赖
npm install

# 构建生产版本
npm run build
```

### 步骤3: 配置 API 地址

创建 `.env.production` 文件：

```bash
VITE_ORANGE_PI_API=http://192.168.31.159:8000
```

或者在构建前修改 `api.ts`。

### 步骤4: 部署前端（两种方式）

#### 方式A: 使用 Vite Preview（简单）

```bash
# 构建后，使用 preview 模式
npm run build
npm run preview -- --host 0.0.0.0 --port 3000
```

然后访问：`http://192.168.31.159:3000`

#### 方式B: 使用 Nginx（推荐生产环境）

```bash
# 安装 Nginx
sudo apt install nginx -y

# 配置 Nginx
sudo nano /etc/nginx/sites-available/sleep-system
```

Nginx 配置内容：
```nginx
server {
    listen 80;
    server_name 192.168.31.159;

    root /home/orangepi/Desktop/theme1/ui/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # 代理 API 请求到后端
    location /api {
        proxy_pass http://localhost:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/sleep-system /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

然后访问：`http://192.168.31.159`

## 常见问题

### Q: 前端显示空白或只有黑色？
A: 检查：
1. 浏览器控制台是否有错误（F12）
2. API 地址是否正确
3. 后端 CORS 是否配置正确（已配置）

### Q: CORS 错误？
A: 后端已经配置了 CORS，允许所有来源。如果还有问题，检查：
- 后端是否正常运行
- 防火墙是否阻止了请求

### Q: 前端资源加载失败？
A: 检查：
1. 网络连接是否正常
2. CDN 资源是否能访问（Tailwind CSS, React 等）
